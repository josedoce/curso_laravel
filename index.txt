o que é o laravel ? um framework php
em que arquitetura é baseado ? mvc
m = model reponsabilidade de troca de dados entre a aplicação e o banco de dados
v = views = responsabilidade de interação com o usuario
c = controllers = reponsabilidae logica da aplicação com finalidade em entregar funções ao usuario 

o que se faz com o laravel? sites com milhares de rotas de forma otimizada e com codigo limpo.

Obs: Pastas a serem ignoradas, vendor, tests

pastas importantes e suas funções

  1-public, onde fica o index.php o unico script que o usuario terá "acesso"
  2-routes, onde ficam as rotas do aplicativo web
  3-resources, onde ficam a view e seus scripts, js, linguagens, css
